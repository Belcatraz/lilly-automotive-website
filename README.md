# Lilly Automotive Website

Single-page marketing website for Lilly Automotive, an auto repair shop in Warner Robins, Georgia.

**Live URL:** https://lillyautomotivellc.com

## Quick Start

```bash
# Clone the repository
git clone https://github.com/Belcatraz/lilly-automotive-website.git

# Open in browser (no build required)
# Simply open index.html in your web browser
```

## Tech Stack

- HTML/CSS/JavaScript (single file, no build process)
- Google Fonts (Bebas Neue, Outfit)
- Flatpickr (date picker)
- EmailJS (form submissions)
- GitHub Pages (hosting)

## Development

This is a static website with no build process. Edit `index.html` directly and open it in a browser to see changes.

### Testing Locally

1. Open `index.html` in your browser
2. Test the contact form (requires EmailJS credentials to actually send)
3. Verify responsive design at different viewport sizes

### Deployment

Push to the `main` branch. GitHub Pages automatically deploys changes.
